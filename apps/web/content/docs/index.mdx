# Welcome to Repobox

Repobox is a **self-hosted AI code agent platform** that enables teams to run AI-powered coding assistants on their own infrastructure.

## Features

- **Self-hosted**: Run on your own servers with full control over your data
- **Git Integration**: Works with GitHub, GitLab, and other Git providers
- **Work Sessions**: Manage AI coding sessions with real-time streaming output
- **Secure**: Your code never leaves your infrastructure
